    What even is this??

    ex3 is an experimental utility Discord bot, created in 2022, written in JavaScript with Discord.js and hosted in Los Angeles. It's got 70+ commands, spanning from (primitive) AI image generation to moderation to dumb internet bullshit.

    Roadmap

    Planned future features include:

    • YouTube search + downloads (yes, for "educational" purposes)
    • Server management tools that don't suck
    • More image generation nonsense

    However, ex3 is built with zero data collection. It can't read messages. That's on purpose - these limitations keep the bot lean and respectful, even if they do make some features annoyingly impossible.

    Canceled. ex3 is on the verge of shutting down.
