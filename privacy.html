    • Privacy policy

    In a nutshell: No data is collected about you.

    What's collected?

    The only data collected is a single integer that is incremented each time a command is triggered by anyone.
    It looks like this:

    400

    How is it used?

    In /stats. I also may look at it from time to time to see how the bot is going.

    Data security

    There isn't any data to protect. It's a publicly accessible number.

    Opt-out

    You cannot opt out of this. It's literally an anonymous number.

    Disclaimer

    This bot utilizes many different APIs, which may have their own set of privacy policies. Please consult the /credits command to see which APIs are used.

    Furthermore, avoid sending sensitive information to the bot. Again, nothing is stored on my end, but that may differ for the APIs used. Some commands allow you to force an ephemeral reply.

    If you have any questions, please email me.
